The premise of this project is to randomly generate recipes from an API and calculate 
the time and steps to make the food as well as give substitutions.
The API that we will use is https://spoonacular.com/food-api.
I also what to generate nutrition facts and save recipes in the localy. 
The project will not require an external libraries
